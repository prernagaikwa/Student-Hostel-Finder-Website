## Features

- Student dashboard with search & filter
- Data stored in MySQL database

## How to Run Locally
1. Install XAMPP
2. Copy project folder to:
   `C:\xampp\htdocs\`
3. Start Apache + MySQL from XAMPP
4. Create database `form_data`
5. Import SQL file or create tables
6. Run:
   `http://localhost/Student-Hostel-Finder-Website-main/`
